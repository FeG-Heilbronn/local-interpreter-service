# Installation of IceCast 
IceCast is used as web-radio-server which distributes broadcasted streams to all endpoints

## Installation
- Install with `> apt-get install icecast2`
- Answer prompted questions
- Check if Instance is running with browser `http://IpToServer:8000` (in my case `ipToServer` was `192.168.178.3`)

## Paths
- Configuration file `/etc/icecast2/icecast.xml`
